uvcdat-web-be
=============

####UV-CDAT overarching web back end container
This repo contains the Python Django application that will host the
following apps
* usage
* docs
* uvcdat-site

####getting up and running
    git clone git://github.com/UV-CDAT/uvcdat-web-be.git
    cd uvcdat-web-be    
    ./setup.sh
